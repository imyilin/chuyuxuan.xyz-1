链接样式

链接的样式，可以用任何CSS属性（如颜色，字体，背景等）。特别的链接，可以有不同的样式，这取决于他们是什么状态。

这四个链接状态是：

  • a:link- 正常，未访问过的链接
  • a:visited - 用户已访问过的链接
  • a:hover - 当用户鼠标放在链接上时
  • a:active- 链接被点击的那一刻
就像这样：
![](https://gitee.com/chuyuxuan/tuc/raw/master/record_gif/css1.gif)

当设置为若干链路状态的样式，也有一些顺序规则：

  • a:hover 必须跟在 a:link 和 a:visited后面
  • a:active必须跟在 a:hover后面
